根据能获取到的信息，自动决定展示哪种卡片形态：

能获取到 展示形态 优先级
标题 + 摘要 + Preview 图 + icon 完整版（大图卡片） P0
标题 + 摘要 + icon（无 Preview 图） 无图版（横排 + 大 icon 占位） P0
仅标题 + icon 无图版（隐藏摘要行） P1
什么都获取不到 降级版（icon + 域名 + 链接） P0 兜底
抓取中（< 3s） 骨架屏过渡 P1
抓取超时（> 3s） 直接降级为降级版 P0 兜底

icon 获取优先级：HTML <link rel="icon"> → /favicon.ico → 域名首字母色块兜底
Preview 图获取优先级：og:image → twitter:image → 页面第一张图（需过滤小图）
摘要获取优先级：meta description → og:description → 正文前 120 字截断
